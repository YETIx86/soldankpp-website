What exactly is Soldank++

Soldank++ is a unique 2D movement game released open source in 2024 initiated by Nedi.
The game is a clone of the Soldat game engine rewritten in modern C++ with exactly look and feel.
The primary objective is to replicate all existing core mechanics precisely while expanding upon them.
Furthermore, the project is committed to fully replacing certain legacy features.

What will be different from Soldat

  • from Pascal language to the newest C++ version
  • netcode optimizations
  • new lobby & launcher
  • server scripting with fast and versatile daScript language
  • new admin tool for servers
  • built-in map editor with new features
  • movement will be the main gamemode

Soldank++ is a clone of the Soldat game engine. The name was inspired on helloer's Soldat Remake.
